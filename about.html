MIMUSA Testing Site

About

This website provides the platform to test the MIMUSA Sentiment Analysis model.
With the input of any sentence or paragraphs, the sentiment value of the text body will be computed.

The displayed result range is as follows:

  • Strongly Positive
  • Positive
  • Neutral
  • Negative
  • Strongly Negative
For specifications on how the code works, kindly refer to the publication of the research paper.

Thank you.
